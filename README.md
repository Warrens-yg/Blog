## 欢迎来到GitHub Pages

您可以使用[GitHub上的编辑器](https://github.com/Warrens-yg/Blog/edit/master/README.md)来维护和预览Markdown文件中网站的内容。

每当您提交到该存储库时，GitHub Pages都会运行[Jekyll](https://jekyllrb.com/)从Markdown文件中的内容重建站点中的页面。

### Markdown

Markdown是一种轻巧且易于使用的语法，可用于样式化文字。它包括以下约定

```markdown
Syntax highlighted code block

# Header 1
## Header 2
### Header 3

- Bulleted
- List

1. Numbered
2. List

**Bold** and _Italic_ and `Code` text

[Link](url) and ![Image](src)
```

有关更多详细信息，请参见[GitHub Flavored Markdown](https://guides.github.com/features/mastering-markdown/)。

### Jekyll主题

您的Pages网站将使用您在[存储库设置](https://github.com/Warrens-yg/Blog/settings)中选择的Jekyll主题的布局和样式。该主题的名称保存在Jekyll _config.yml配置文件中。
